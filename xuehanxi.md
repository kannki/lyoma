# 1.程序简介
选择暑假预习作业选题1：学生学分绩点计算工具，使用C++语言实现对数据库中部分电气学生大二下成绩学分绩点进行运算的操作。  
# 2.主要内容
## 1.学生学分绩点数据库
   使用SQL Server Management Studio
## 2.实现功能
增加成绩  
删除成绩  
修改成绩  
查询成绩
## 3.包含科目
信号与系统  
电磁场   
电机学  
加权成绩
# 3.操作步骤
## 1.增加成绩
选择增加成绩  
输入学生姓名  
按*信号与系统 电磁场 电机学*的顺序逐个输入成绩  
成绩录入完成
## 2.删除成绩
选择删除成绩  
输入学生姓名  
成绩删除完成  
## 3.修改成绩
选择修改成绩  
按*信号与系统 电磁场 电机学*的顺序逐个输入成绩   
成绩修改完成
## 4.查询成绩
选择查询成绩  
按***信号与系统 电磁场 电机学 总加权***的顺序逐个显示成绩   
成绩查询完成
